## 🧠 Project Overview

This project transforms raw PhonePe data into a fully interactive dashboard that enables users to explore:

- State-wise transaction distribution

- User growth trends across years

- Insurance adoption patterns

- Drill-down insights at state level

The goal is to simulate a **real-world decision-support system** using data visualization and analytics.

---

## 📊 Key Features

- 📍 State-wise transaction heatmap (India map)

- 📈 Year-over-Year growth analysis

- 🏆 Top performing states and brands

- 🔍 State-level drilldown analytics

- 📊 Transactions, Users & Insurance modules

- 🌙 Dark mode toggle

- 📥 Downloadable dataset

---

## 📊 Business Insights

- Maharashtra, Karnataka, and Tamil Nadu contribute the highest transaction volumes, indicating strong digital payment penetration.

- Digital transactions show consistent year-over-year growth, especially post-2019.

- User adoption is dominated by a few major brands, suggesting limited market fragmentation.

- Insurance adoption is growing but remains lower compared to transaction activity, indicating potential market expansion.

---

## 🏗️ Architecture

CSV Data → Pandas Processing → Plotly Visualizations → Streamlit Dashboard → Deployed on Streamlit Cloud
